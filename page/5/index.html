go 标准库之 strings

发表于 2019-06-09 | 分类于 golang

字符串方法

方法概览

函数 作用
func Contains(s, substr string) bool 检测 substr 是否在 s 中存在
func Compare(a, b string) int 将字符串按照字母表相比较返回整数，a<b则为-1，a>b则为+1，a==b则为0
func Count(s, substr string) int 返回 substr 在 s 中存在的次数
func EqualFold(s, t string) bool 不区分大小写的完全匹配
func Fields(s string) []string 以连续的空白字符为分隔符（\t, \n, \v, \f, \r, ‘ ‘, U+0085 (NEL), U+00A0 (NBSP)），将 s 分割成多个字串，结果中不包含空白本身
func HasPrefix(s, prefix string) bool 测试 s 是否以 prefix 开头
func HasSuffix(s, suffix string) bool 测试 s 是否以 suffix 结尾
func Index(s, substr string) int 返回 s 中第一个匹配 substr 的索引，如果不存在则返回 -1
func LastIndex(s, substr string) int 返回 s 中最后一个匹配 substr 的索引，如果不存在返回 -1
func Join(a []string, sep string) string 指定分隔符连接一个数组中的元素以创建成一个字符串
func Replace(s, old, new string, n int) string 在 s 这个字符串中查找替换字符串，并且指定替换次数，0 不替换，-1 全部替换
func Split(s, sep string) []string 使用分隔符对字符串进行切分，返回一个切片
func SplitN(s, sep string, n int) []string 使用分隔符对字符串进行指定次数的切分，返回一个切片
func SplitAfter(s, sep string) []string 使用分隔符对字符串进行切分，返回一个切片，但保留每个切片的分隔符结尾
func SplitAfterN(s, sep string, n int) []string 使用分隔符对字符串，按指定次数进行切分，返回一个切片，但保留每个切片的分隔符结尾
func Title(s string) string 返回首字母大写的字符串
func ToLower(s string) string 将字符串转换为小写
func ToUpper(s string) string 将字符串全部转换为大写
func TrimSpace(s string) string 从开头和结尾删除空白
func TrimSuffix(s, suffix string) string 从结尾删除匹配的字符串
func TrimPrefix(s, prefix string) string 删除开头的指定字符串

字符串检测

Contains 是否包含字符串

Compare 按字母表比大小

Count返回指定字符串存在的次数

EqualFold不区分大小写是否匹配

HasPrefix测试是否以指定字符串开头

HasSuffix测试是否以指定字符串结尾

字符串索引位置

Index返回第一个匹配字串的索引

LastIndex返回最后一个匹配字串的索引

切片和连接

Join将切片连接为字符串

Fields用空格字符分隔返回切片

Split指定分隔字串对字符串进行切片

SplitN指定分隔字串对字符串进行指定次数切片

SplitAfter指定分隔符字串进行切片，保留分隔符

SplitAfterN指定分隔符字串指定指定次数进行切片，保留分隔符

大小写转换

Title字符串首字母大写

ToLower全部转为小写

ToUpper全部转为大写

删除开头结尾查找替换

Replace查找替换字符串

TrimSpace从开头和结尾删除空白

TrimPrefix从字符串开头删除匹配的任意单个字符

TrimSuffix从结尾删除匹配到的字符串

方法概览

函数 作用
func ContainsAny(s, chars string) bool 检测 chars 中的任意 Unicode码位值是否在 s 内
func ContainsRune(s string, r rune) bool 检测一个 rune 这个 Unicode码位值是否在 s 内存在

func IndexAny(s, chars string) int 返回 s 中第一个匹配 chars 中任意字符的索引位置
func IndexByte(s string, c byte) int 返回 s 中第一个匹配 byte （英文字符）的索引位置

func LastIndexAny(s, chars string) int 返回 s 中最后一个匹配 chars 中任意字符的索引位置
func LastIndexByte(s string, c byte) int 返回 s 中最后一个匹配 byte （英文字符）的索引位置
func LastIndexFunc(s string, f func(rune) bool) int 返回 s 中最后一个符合 f 这个函数的字符位置

func ToTitleSpecial(c unicode.SpecialCase, s string) string 按照指定的字符集映射为大写
func ToLowerSpecial(c unicode.SpecialCase, s string) string 按照指定的字符集映射为小写
func ToUpperSpecial(c unicode.SpecialCase, s string) string 将字符串按照指定的字符集转换为大写
func Trim(s string, cutset string) string 从字符串开头和结尾删除 cutset 这个字符集合中的任意单个字符
func TrimFunc(s string, f func(rune) bool) string 函数决定如何删除字符串开头和结尾的字符
func TrimRight(s string, cutset string) string 删除结尾的指定字符集合中的任意字符
func TrimRightFunc(s string, f func(rune) bool) string 函数决定如何从结尾删除字符
func TrimLeft(s string, cutset string) string 删除开头的指定字符集合中的任意字符
func TTrimLeftFunc(s string, f func(rune) bool) string 函数决定如何从开头删除字符

ContainsAny检测是否包含任意字符

ContainsRune检测是否包含指定Unicode码位值

FieldsFunc使用字符由外置函数决定切片

字符索引位置

IndexAny返回第一个匹配任意字符的索引

IndexByte返回第一个匹配byte字符的索引

IndexFunc返回第一个函数判定为true的索引

IndexRune返回第一个匹配rune字符的索引

LastIndexAny返回最后一个匹配任意字符的索引

LastIndexByte返回最后一个匹配byte字符的索引

LastIndexFunc返回最后一个函数片顶为true的索引

映射字符大小写

Map函数决定映射字符串

ToTitle全部转为大写

ToTitleSpecial按照指定的字符集映射为大写

ToLowerSpecial按照指定字符集全部映射为小写

ToUpperSpecial按照指定字符集全部映射为大写

删除开头结尾

Trim从字符串开头和结尾删除匹配的任意单个字符

TrimFunc函数决定删除哪些开头和结尾的字符

TrimRight删除结尾的指定字符集合中的任意字符

TrimRightFunc函数决定如何从结尾删除字符

TrimLeft删除开头的指定字符集合中的任意字符

TrimLeftFunc函数决定如何从开头删除字符

时间类型

时间类型在实际开发中使用频率很高，代表现实世界的时间

Go 语言中时间类型不是关键字，而是使用 time 包下的 Time 结构体

格式常量

这些是在 Time.Format 和 time.Parse 中使用的预定义常量，需要注意的是无论格式如何变，模板必须为：2006 年 1 月 2 日 15 点 04 分 05 秒，快捷记忆：2016/1/2/3/4/5 即可。

Sleep睡眠

Go 语言中 main() 函数为主线程（协程），程序是从上向下执行的，可以通过 time.Sleep(n) 设置让程序阻塞多少纳秒

Duration类型

Duration 类型代表两个时间点之间经过的时间，以纳秒为单位。可表示的最长时间段大约290年。

func Since(t Time) Duration 返回从 t 到现在经历了多久，是：time.Now().Sub(t) 的缩写
func Until(t Time) Duration 返回到 t 需要多久，是 t.Sub(time.Now()) 的缩写

Location类型

Location 将时间映射到当前的时区，通常表示在地理区域中使用的时间偏移集合。

Local表示系统的本地时区。

UTC表示通用协调时间（UTC）。

函数 作用
func FixedZone(name string, offset int) *Location 给定名称和偏移量返回 Location
func LoadLocation(name string) (*Location, error) 返回使用给定名字创建的 Location，如果名称为””或”UTC”，则返回UTC，如果名称为”Local”，则返回Local，其他视作 从 IANA 时区数据库文件中对应的位置名称，例如：Asia/Shanghai。
func LoadLocationFromTZData(name string, data []byte) (*Location, error) 接收时区名称和时区数据文件，格式必须是：/etc/localtime， 返回一个 Location。

FixedZone给定名称和偏移量创建时区

LoadLocation根据名称创建时区

String返回一个时区对象的字符串表示

Unix从时间戳解析时间

Date从数字制作时间

Parse指定格式将字符串解析为时间

Parse 将时间解释为 UTC; 当给定区域偏移或缩写时，Parse 会尝试将其与本地位置匹配;

ParseInLocation指定格式将字符串解析为时间，备选时区信息

与 Parse 一样，但在没有时区信息的情况下，ParseInLocation 将时间解释为给定位置，当给定时区时，ParseInLocation 使用给定的位置。
